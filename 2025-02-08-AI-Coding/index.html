    Finding LLM-Shaped Problems (The Jagged Frontier)

    The real skill isn't prompt crafting it's recognising which problems are LLM-shaped. LLMs are shockingly good at some things and shockingly bad at others, and the boundary is jagged and non-intuitive.

    • One Useful Thing (Ethan Mollick's Substack) free, ongoing. Every post explores what is/isn't LLM-shaped. Wharton professor, research-backed.
    • Simon Willison 2025: The Year in LLMs comprehensive annual review of what changed, what works, what doesn't.
    • Simon Willison LLM Predictions for 2026
    • Simon Willison How StrongDM's AI team build serious software without even looking at the code non-interactive "software factory" where specs drive agents. Feb 2026.
    • LLMs as a Lossy Encyclopedia great mental model: LLMs shine at processing/understanding/adapting information, not precise retrieval.
    • AI tools for software engineers, but without the hype Simon Willison

    Context Engineering (Beyond Prompt Engineering)

    The shift from crafting clever prompts to engineering the input environment giving the model everything it needs to succeed. This is what pi skills do.

    • Addy Osmani's Substack (Elevate) free. LLM coding workflows, context engineering, agentic development. 600K readers.

    Understanding Agent Architecture

    Agents are simpler than the hype suggests: it's a loop with inputs and outputs from an environment where it can call tools to augment itself. You don't need to build agents from scratch just understand the loop well enough to use agent tools effectively and not get suckered by hype.

    • Anthropic Building Effective Agents canonical reference. Progression from prompt chaining routing tool use autonomous loop. Also: walkthrough video, overview video.
    • AI Agents from First Principles (Cameron R. Wolfe) builds up from basic LLM concepts to agents. Always free. Excellent.
    • How AI Agents Really Work: Beyond the Hype, Inside the Architecture what's actually happening under the hood.
    • Building Robust Agentic AI: Why Simplicity and Observability Matter More Than Cleverness simple loops with good feedback beat clever architectures.
    • OpenClaw Explained (CJ Avilla) demystifies why AI agents feel "alive". It's inputs, queues, and a loop.
    • Chess tools built with Claude 61-year-old non-programmer builds real tools through conversational coding.
